\section{Введение и постановка задачи}

Семантическая сегментация снимков дистанционного зондирования Земли (ДЗЗ) является
одной из ключевых задач геоинформатики. Её применяют для мониторинга землепользования,
планирования городской инфраструктуры, оценки ущерба от стихийных бедствий и точного
земледелия. Традиционные методы сегментации требуют большого объёма вручную размеченных
данных, получение которых для спутниковых снимков трудозатратно и дорогостояще:
эксперты-интерпретаторы должны работать с высокоразрешающими мультиспектральными
изображениями, охватывающими площади в сотни квадратных километров.

\textbf{Цель работы} — разработать систему семантической сегментации снимков ДЗЗ,
не требующую ни одной размеченной выборки на этапе обучения. Для этого используется
комбинация трёх взаимодополняющих техник. \textbf{Трансферное обучение} на основе
предобученного визуального трансформера DINOv2~\cite{oquab2024dinov2} (Meta AI, 2023),
обученного методом самообучения на 142~млн изображений, обеспечивает богатые
семантические признаки без дополнительной разметки. \textbf{Кластеризация признаков}
через MiniBatch K-means инициализирует начальные псевдометки непосредственно из
DINOv2-представлений. \textbf{Self-training с Mean Teacher}~\cite{tarvainen2017mean}
итеративно уточняет псевдометки, обучая student-модель согласовываться с teacher-моделью,
веса которой обновляются как экспоненциальное скользящее среднее (EMA) весов student.

\begin{infobox}[Датасет]
  \textbf{LoveDA}~\cite{wang2021loveda} (Land-cOVEr Domain Adaptive semantic segmentation)
  распространяется через Kaggle и библиотеку torchgeo. Содержит высокоразрешающие
  (0.3\,м/пиксель) RGB-снимки городской и сельской местности Китая с пиксельной
  разметкой по 7 классам. Общий объём обучающей и валидационной выборок — 4\,191
  изображение размером $1024 \times 1024$ пикселей.
\end{infobox}

\subsection{Предобработка}

Изображения масштабируются до фиксированного размера (224 или 448\,пикселей по стороне)
и нормализуются по статистике ImageNet: $\mu = (0.485, 0.456, 0.406)$,
$\sigma = (0.229, 0.224, 0.225)$. Размер выбирается кратным 14 --- размеру патча
DINOv2 ViT, что обеспечивает целочисленное разрешение карт признаков.
Разметка при обучении \textbf{не используется}; маски GT применяются исключительно
для финальной оценки через венгерский алгоритм.

\section{Методология}

\subsection{Архитектура системы}

\begin{figure}[htbp]
  \centering
  \includegraphics[width=0.88\textwidth]{fig5_pipeline.png}
  \caption{Пайплайн несупервизируемой сегментации. Слева направо: обработка входного
           изображения DINOv2 backbone, декодирование SegFormer, трёхфазное обучение
           (K-means $\to$ warm-up $\to$ self-training с Mean Teacher).}
  \label{fig:pipeline}
\end{figure}

Система состоит из двух компонентов, объединённых в пару student/teacher:

\textbf{DINOv2 ViT Backbone.} Принимает RGB-изображение размером $H \times W$ и
извлекает признаки четырёх промежуточных слоёв трансформера, каждый с пространственным
разрешением $H/14 \times W/14 \times D$ (для ViT-S/14: $D = 384$). При замороженном
backbone обучается только декодер; с раунда 3 размораживаются последние четыре
блока трансформера.

\textbf{SegFormer MLP Decoder~\cite{xie2021segformer}.} Четыре независимых MLP
проецируют признаки каждого масштаба в пространство размерности \texttt{decoder\_dim}
(по умолчанию 256). Все карты признаков билинейно интерполируются до размера наибольшей
карты, конкатенируются и сливаются свёрткой $1{\times}1$ + BN + ReLU. Финальная
свёртка $1{\times}1$ выдаёт тензор логитов формы \texttt{num\_classes} $\times H \times W$.

\subsection{Трёхфазный пайплайн обучения}

\subsubsection{Фаза 0 --- Инициализация псевдометок (K-means)}

Замороженный backbone извлекает патч-признаки последнего слоя для всех обучающих снимков.
Все $N_\text{img} \times H_p \times W_p$ векторов (где $H_p = W_p = H/14$) подаются в
MiniBatch K-means с $k{=}7$ кластерами. Каждому патчу присваивается кластерный ID,
образуя начальную псевдометку с разрешением $H_p \times W_p$.

\subsubsection{Фаза 1 --- Разогрев (Warm-up)}

Декодер обучается на K-means псевдометках при замороженном backbone стандартной
кросс-энтропией по всем пикселям. Цель фазы --- привести декодер в разумное начальное
состояние перед итеративным самообучением.

\subsubsection{Фаза 2 --- Self-training раунды}

Каждый раунд включает четыре этапа. \textbf{Обновление псевдометок:} teacher-модель
прогоняет все обучающие снимки и сохраняет предсказания для пикселей, где максимальная
softmax-вероятность превышает порог $\theta$. \textbf{Обучение student:} teacher
получает слабо аугментированные снимки (случайное горизонтальное отражение) и формирует
мягкие псевдометки с маской уверенности; student получает сильно аугментированные снимки
(случайный crop + colour jitter + Gaussian blur). Суммарная функция потерь:

\begin{equation}
  \mathcal{L} = \mathcal{L}_\text{CE}(s_\theta, \hat{y}, \mathbf{1}) +
                0.5 \cdot \mathcal{L}_\text{KL}(s_\theta, t_\theta, \mathbf{m}_\text{conf}),
  \label{eq:loss}
\end{equation}

\noindent где $\mathcal{L}_\text{CE}$ --- кросс-энтропия по всем пикселям,
$\mathcal{L}_\text{KL}$ --- дивергенция Кульбака--Лейблера только по уверенным пикселям
($\mathbf{m}_\text{conf}$). \textbf{EMA-обновление teacher:} после каждого шага

\begin{equation}
  \theta_T \leftarrow \alpha \cdot \theta_T + (1 - \alpha) \cdot \theta_S, \quad \alpha = 0.999.
  \label{eq:ema}
\end{equation}

\noindent \textbf{Снижение порога:} каждый раунд $\theta$ уменьшается на $\Delta\theta = 0.025$,
но не ниже $\theta_\text{min} = 0.70$, постепенно включая менее уверенные предсказания.

\subsection{Оценка качества}

Поскольку кластерные ID произвольны и не совпадают с семантическими классами,
для оценки применяется венгерский алгоритм (Hungarian matching / linear assignment),
находящий оптимальную перестановку кластеров:

\begin{equation}
  \text{mIoU} = \frac{1}{C} \sum_{c=1}^{C} \text{IoU}\!\left(\sigma^*(c), c\right),
  \label{eq:miou}
\end{equation}

\noindent где $\sigma^*$ --- оптимальная перестановка, $C = 7$. Это единственный
корректный способ измерить качество несупервизированной сегментации.

    \item Порог $\theta = 0.70$ слишком высок для 5 эпох: 0\,\% пикселей
          признаны уверенными, поэтому consistency loss $= 0$ и обучение идёт
          исключительно по K-means псевдометкам.
    \item mIoU 9.86\,\% --- базовая линия для быстрого прогона; при полном
          обучении (5+ раундов, $\theta{:}\;0.95{\to}0.70$, 448\,px, ViT-B/14)
          ожидается 35--45\,\%.
  \end{itemize}
\end{resultbox}

\subsection{Анализ ошибок и ограничения}

\begin{warnbox}[Ограничения текущего запуска]
  \textbf{Confidence collapse.} При $\theta = 0.70$ с 5 эпохами — 0\,\% уверенных
  пикселей. Для пробного запуска рекомендуется $\theta = 0.3$--$0.5$.

  \textbf{Фон против сельхозугодий.} Наиболее частая путаница из-за схожей текстуры
  при разрешении 224\,px.

  \textbf{Редкий класс Barren} (4.7\,\% пикселей) сложен для кластеризации --- самый
  низкий IoU.

  \textbf{Дороги} как тонкие линейные структуры плохо выделяются при 224\,px;
  при 448\,px качество значительно выше.
\end{warnbox}

\section{Заключение}

\subsection{Итоги работы}

Разработан и верифицирован полный пайплайн несупервизированной семантической
сегментации снимков ДЗЗ. \textbf{Принципиальное преимущество} предложенного подхода
состоит в том, что он не требует ни одной размеченной выборки на этапе обучения:
семантическая информация извлекается исключительно из структуры данных через
DINOv2-признаки и итеративное самообучение.

\textbf{K-means инициализация} (645\,632 патч-вектора, $d = 384$) создаёт
семантически состоятельную начальную разбивку уже за несколько секунд. Механизм
\textbf{Mean Teacher} с EMA ($\alpha = 0.999$) обеспечивает стабильную генерацию
псевдометок: teacher накапливает исторические знания, student обучается на жёстких
аугментациях. \textbf{Адаптивный порог уверенности} постепенно расширяет множество
учитываемых пикселей по мере роста качества teacher. \textbf{Венгерский алгоритм}
при оценке корректно выравнивает произвольные кластерные ID с семантическими классами
без дополнительного supervision.

\subsection{Ключевые вклады}

\begin{enumerate}
  \item Интеграция DINOv2 + SegFormer для несупервизированной сегментации ДЗЗ.
  \item Трёхфазный пайплайн: K-means $\to$ warm-up $\to$ итеративное self-training.
  \item Верификация на реальном датасете LoveDA (RTX~5070~Ti, CUDA~13.2).
  \item Публичная кодовая база с воспроизводимыми экспериментами:
        \texttt{make env \&\& make download \&\& make train-gpu}.
\end{enumerate}

\subsection{Направления дальнейшей работы}

\textbf{Слабая разметка (semi-supervised):} добавление всего 1--5\,\% размеченных
снимков ожидаемо даст скачок mIoU на $+10$--$15\,\%$. \textbf{Мультиспектральные
данные Sentinel-2} (13 каналов): реализованный \texttt{ChannelAdapter} ($13{\to}3$)
готов к тестированию. \textbf{Масштаб разрешения} --- переход на 448\,px с ViT-B/14
даёт $+5$--$8\,\%$ mIoU. \textbf{Knowledge distillation:} после обучения teacher
можно дистиллировать в компактную модель для инференса на устройстве без GPU.
\textbf{Temporal consistency:} для серий снимков одного региона временна́я когерентность
служит дополнительным несупервизированным сигналом.
